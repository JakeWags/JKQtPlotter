34/** \brief This is a base-class for all bar graphs with vertical or horizontal orientation (the orientation is implemented in dervied classes!)
35 * \ingroup jkqtplotter_barssticks
36 *
37 * This class plots a bargraph. This image explains the parameters:
38 *
39 * \image html bargraph_basics.png
40 *
41 * By default the sift parameter is, so the bar is centered at the x-value. The width is 0.9,
42 * so adjacent bars are plotted with a small distance between them. It is possible to use these two parameters
43 * to plot multiple bars for every x-value, by having on JKQTPSpecialLineHorizontalGraph object per
44 * set of bars that belong together. For example for three bars per x-value one would set:
45 * \verbatim
46 * width=0.3
47 * shift=-0.3 / 0 / +0.3
48 * \endverbatim
49 * This results in a bargraph, as shown here:
50 *
51 * \image html JKQTPBarVerticalGraph.png
52 *
53 * You can also set FillMode::TwoColorFilling, which uses different fill styles for bars above and below
54 * the baseline of the graph:
55 *
56 * \image html JKQTPBarVerticalGraphTwoColorFilling.png
57 *
58 * \see JKQTPBarHorizontalGraph, JKQTPBarVerticalGraph
59 */
60class JKQTPLOTTER_LIB_EXPORT JKQTPBarGraphBase: public JKQTPXYBaselineGraph, public JKQTPGraphLineStyleMixin, public JKQTPGraphFillStyleMixin {
