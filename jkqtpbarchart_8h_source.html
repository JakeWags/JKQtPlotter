36/*! \brief This implements a bar graph with bars starting at \f$ yoverride \f$ to \f$ y=f(x) \f$
37 \ingroup jkqtplotter_barssticks
38
39 This class plots a bargraph. This image explains the parameters:
40
41 \image html bargraph_basics.png
42
43 By default the sift parameter is, so the bar is centered at the x-value. The width is 0.9,
44 so adjacent bars are plotted with a small distance between them. It is possible to use these two parameters
45 to plot multiple bars for every x-value, by having on JKQTPSpecialLineHorizontalGraph object per
46 set of bars that belong together. For example for three bars per x-value one would set:
47 \verbatim
48 widthoverride.3
49 shift=-0.3 / 0 / +0.3
50 \endverbatim
51 This results in a bargraph, as shown here:
52
53 \image html plot_bargraphverplot.png
54
55
56 You can use JKQTPlotter::addHorizontalBargraph() to add a series of bargraphs, where the width and shift are determined
57 automatically. The y-columns are given as a QVector<int> to this function.
58
59 \see JKQTPBarHorizontalGraph, \ref JKQTPlotterBarcharts, jkqtpstatAddHHistogram1D(), jkqtpstatAddHHistogram1DAutoranged()
60 */
